# Re-Edit

- [x] Text Formatting
- [x] Selectable font size and family
- [x] Images and Videos (gifs)
- [x] Links
- [x] Lists
- [x] Code blocks
- [x] Tables
- [x] Custom highlighting rules
- [x] Realtime Collaboration
